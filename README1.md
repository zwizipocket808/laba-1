## Задание 1: Расстояния между городами (00_distance.py)

**Условие задачи:**  
Дан словарь с координатами трёх городов. Нужно создать словарь словарей расстояний между каждой парой городов.

**Почему я так решил:**  
Нужно сделать вложенный словарь. Для каждого города создать внутренний словарь, потом для каждой пары городов посчитать расстояние по формуле √((x₁-x₂)² + (y₁-y₂)²).

**Как решил:**  
Внешний цикл по городам создаёт запись во внешнем словаре. Внутренний цикл по городам заполняет внутренний словарь. Если город тот же самый, расстояние 0, иначе считаю по формуле. Результаты округлил до 2 знаков.

## Задание 2: Геометрия (радиус круга) (01_circle.py)

**Условие задачи:**  
Есть радиус круга 42. Нужно вычислить площадь этого круга с точностью до 4 знаков после запятой. Затем есть две точки: (23, 34) и (30, 30). Нужно определить, лежат ли они внутри круга с центром в начале координат.

**Почему я так решил:**  
Для площади круга я использовал формулу πr². Число π взял 3.1415926 как указано в задании. Чтобы узнать, лежит ли точка в круге, нужно сравнить расстояние от точки до центра с радиусом. Если расстояние меньше или равно радиусу, точка внутри. Расстояние считается по формуле √(x² + y²) для центра в (0,0). Результат округлил до 4 знаков как просили.

**Как решил:**  
Просто подставил числа в формулы и вывел результаты. Сначала посчитал площадь, потом для каждой точки расстояние и сравнил с радиусом.

# ___Шпаргалка Git для быстрого копирования___

## Базовая настройка
`git config --global user.name "Твое Имя"`
`git config --global user.email "твоя@почта.com"`

## Начало работы
`git init` - создать репозиторий
`git clone ссылка` - скопировать чужой репозиторий
`git status` - посмотреть что изменилось

## Сохранение изменений
`git add .` - добавить все файлы
`git add файл.py` - добавить один файл
`git commit -m "сообщение"` - сохранить изменения
`git log` - посмотреть историю

## Ветки
`git branch` - показать ветки
`git branch имя` - создать ветку
`git checkout имя` - переключиться на ветку
`git checkout -b имя` - создать и переключиться
`git merge ветка` - объединить ветки

## Работа с GitHub
`git remote add origin ссылка` - подключить GitHub
`git push origin main` - отправить на GitHub
`git pull origin main` - скачать с GitHub
`git fetch` - проверить изменения

## Отмена действий
`git checkout -- файл.py` - отменить изменения в файле
`git reset --soft HEAD~1` - отменить последний коммит
`git stash` - временно спрятать изменения
`git stash pop` - вернуть спрятанное

## Полезные команды
`git diff` - посмотреть изменения
`git rm файл` - удалить файл
`git mv старый новый` - переименовать
`git tag v1.0` - создать версию
`git push --tags` - отправить теги

## Короткие команды
`git add .` → `git commit -m "описание"` → `git push origin main`
Это три главные команды для работы каждый день.
